# FRACTAL//STUDIO — Portfolio

Personal portfolio site for game design, 3D modeling, texturing, and sound design work.

## Stack

- **HTML / CSS / JS** — No frameworks, no build step
- **Fonts** — Space Grotesk, JetBrains Mono, Archivo Black (Google Fonts)
- **Style** — Acid-graphic aesthetic with film grain overlay

## Structure

```
index.html    — Single-page portfolio
styles.css    — All styles (responsive, reduced-motion support)
script.js     — Intersection observer reveals, play toggles, micro-interactions
```

## Local Development

Open `index.html` in a browser. No build tools required.

For a local server with live reload, use any static server:

```bash
npx serve .
```

## Deployment

This is a static site — deploy to GitHub Pages, Netlify, Vercel, or any static host.
